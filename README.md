# pedidos
Projeto pessoal para fazer pedidos para revenderores de produtos por folhetos AVON ou JEQUETI.

-	Com esse sistema é possivel criar planilhas separadas por campanha e nome das revendedoras.
-	Com a planilha serár possivel preencher com os produtos, quantidades e preço.
-	Somando o total dos produtos da companha atual.
-	Pesquisar campanhas passadas, que ficam salvas no banco de dados.
-	Editar planilhas de campanhas especificas.
-	E gerar pdf's com a planilha para uma futura impressão.
